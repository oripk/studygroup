자바에는 compiler 와 interpreter가 있다.
compiler 는 우리가 쓰는 언어에서 컴퓨터가 쓰는 언어로 바꿔주는 기능을 한다.
interpreter은 compiler와 비슷한 역활을 하지만 즉시 반영해서 보여준다.
Java는 Bytecode라고 불리는 언어로 바꿔주는 기계가 있어야한다. 그것을 Java Virtual Machine(JVM)이라 말한다.
한번 바꾸게 되면 다시 바꿀 필요가 없다.
Java program -> Java compiler -> Bytecode program -> Bytecode interpreter(JVM) ->
		 Machine -language instructions -> Computer execution of machine-language instructions -> out put of java program

Data for Java program --------------->  Computer execution of machine-language instructions -> out put of java program
